Quick Notes

  • Modules, Component, Directives, Services, Pipes, Routing

  • Modules are the fundamental building blocks of ang app, they are class with the NgModule decorator

  • Component are like views, a set of screen element with which user interact, using decorator, @Component

  • Component- event bindings, property bindings

  • Directives transfer element of HTML templates

  • Structural directives are those that alter the HTML tags or the DOM of page

  • Attribute directives are those that change the appearance or behavior of existing elements in our template

  • Service is using to share data or logic that is not specifically associated with a component

  • Pipes allow us to transform data from one format to another

  • Routing is used to have navigation functionality in our angular application
